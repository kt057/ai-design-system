# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works with multi-package repos, or single-package repos to help you version and publish your code. You can find the full documentation for it [in the changesets repo](https://github.com/changesets/changesets).

## Workflow

1. Make your code changes.
2. Run `pnpm changeset` and pick a bump type (`patch` / `minor` / `major`) and write a one-line summary.
3. Open a PR — the changeset file is committed alongside your code.
4. When the PR merges to `main`, the `release` workflow opens (or updates) a "Version Packages" PR that bumps the version and writes the changelog.
5. Merging that PR triggers `pnpm release`, which runs `pnpm build` and then `changeset publish` to push to npm.
